01. Fraction-Percentage Foundational Conversions

SSC MTS, GD, TNPSC Group 4 Frequency: High (Base Skill)

How to Identify This Pattern

Direct query asking to parse a non-standard fraction, compound fractional mixed-number, or long decimal expression into its pure percentage form, or the exact reverse.

Three Solving Approaches

a) Traditional Method

To convert fraction to percentage, multiply by 100 and add the symbol %. For decimal conversion, shift decimal place two locations rightward. Formula balance: Value = (Numerator / Denominator) x 100.

b) Traditional Shortcut Method

Break up complex fractions into recognizable linear additions of base reciprocal components whose fractional conversions are fully memorized (e.g., 5/6 = 1 - 1/6).

c) Direct Shortcut/Trick

Use base scaling: If 1/9 = 11.11%, then 4/9 = 4 x 11.11% = 44.44%. Memorize common mixed fractions configurations directly.

Step-by-Step Approach Summary

  1. Isolate the raw integer component if dealing with an improper fraction.
  2. Map the residual fractional tail to a standard fraction table value.
  3. Multiply the mapped standard percentage with the corresponding multiplier scale factor.

Solved Examples

Q1 [Easy]: Convert 3/8 into its exact percentage equivalent.

Shortcut Solution: We know 1/8 = 12.5%. Therefore, 3/8 = 3 x 12.5% = 37.5%.

Final Answer: 37.5%

Q2 [Easy]: Convert 0.0625 into its percentage notation.

Shortcut Solution: Shift decimal point two spaces to the right: 0.0625 x 100 = 6.25%. Alternatively, recognize 0.0625 = 1/16 = 6.25%.

Final Answer: 6.25%

Q3 [Moderate]: Convert 13/7 to its exact percentage form.

Traditional Solution: (13 / 7) x 100 = 1300 / 7 = 185.714%.

Shortcut Solution: 13/7 = 2 - 1/7 = 200% - 14.28% = 185.72%.

Final Answer: 185.71% (or 185 and 5/7 %)

Q4 [Moderate]: What is 44.44% of 630?

Shortcut Solution: Convert 44.44% to its fractional base: 4 x 11.11% = 4 x (1/9) = 4/9. Now calculate (4/9) x 630 = 4 x 70 = 280.

Final Answer: 280

Q5 [Hard/Twisted]: Express 83.33% as a fraction and use it to find 83.33% of 1728.
[TRAP] Do not attempt long multiplication with 0.8333 x 1728.

Traditional Solution: (83.33 / 100) x 1728, which becomes algebraically messy due to repeating digits.

Shortcut Solution: Recognize 83.33% = 100% - 16.67% = 1 - 1/6 = 5/6. Then compute (5/6) x 1728 = 5 x 288 = 1440.

💎 Live Gem: Recognizing percentages relative to 100% bounds saves vital baseline computation seconds.

Final Answer: 1440

02. Single Linear Percentage Scale Shift

SSC CHSL, MTS, TNPSC Group 4 Frequency: Very High

How to Identify This Pattern

A singular scalar amount is updated. The question states that a value changes from dynamic baseline X to final state Y and demands the net scale alteration in percentage format.

Three Solving Approaches

a) Traditional Method

Apply fundamental delta evaluation formula: Percentage Change = [(Final - Initial) / Initial] x 100. Ensure proper preservation of structural signs.

b) Traditional Shortcut Method

Construct direct numerical scaling ratios. If value transitions from 40 to 50, specify the linear tracking line as 40 : 50 layout which maps to 4 : 5, showcasing a clean single increment on a base structural value of 4.

c) Direct Shortcut/Trick

Fractional Net Delta Strategy: Change = Delta / Initial Base. Instantly transform the final ratio component value into a standardized table lookup vector.

Step-by-Step Approach Summary

  1. Isolate the baseline element ("than X", "compared to X" identifies your initial base).
  2. Find the absolute delta difference between the numbers.
  3. Express as Delta / Base and multiply by 100 or use the fraction table.

Solved Examples

Q1 [Easy]: A commodity price spikes from ₹60 to ₹75. Find the percentage increase.

Shortcut Solution: Ratio = 60 : 75 tracking down to 4 : 5. Increase = 1 part on base 4 = 1/4 = 25%.

Final Answer: 25% Increase

Q2 [Easy]: Production shifts down from 120 units to 90 units. Determine percentage drop.

Shortcut Solution: Ratio = 120 : 90 tracking down to 4 : 3. Drop = 1 part on base 4 = 1/4 = 25%.

Final Answer: 25% Decrease

Q3 [Moderate]: Due to supply constraints, price of an engine increases from ₹8400 to ₹10500. Calculate the change.

Traditional Solution: [(10500 - 8400) / 8400] x 100 = (2100 / 8400) x 100 = (1 / 4) x 100 = 25%.

Shortcut Solution: Cancel hundreds digits: 84 : 105. Divide by 21 to simplify: 4 : 5. Growth structural ratio = 1/4 = 25%.

Final Answer: 25%

Q4 [Moderate]: A worker's hourly wage increased by 12.5%. If he used to earn ₹320 per day, find his new daily earnings assuming identical hours.

Shortcut Solution: 12.5% = 1/8. Growth means old base is 8, new base is 8 + 1 = 9. If 8 units = 320, then 1 unit = 40. New earnings scale = 9 x 40 = 360.

Final Answer: ₹360

Q5 [Hard/Twisted]: The revenue of a firm dropped from ₹14.4 Crores to ₹12.1 Crores. If this historical drop is expressed as a percentage, what value is obtained? Evaluate up to two decimal structures.
[TRAP] Do not compute long division with 2.3 / 14.4. Clean the decimals first.

Traditional Solution: Delta = 14.4 - 12.1 = 2.3. Percentage calculation = (2.3 / 14.4) x 100 = 2300 / 144 = 15.97%.

Shortcut Solution: Scale integers up to 144 : 121. Net Delta = 23. Target setup = (23 / 144) x 100. Note that 10% of 144 = 14.4, 5% is 7.2 (which sets 15% to 21.6). Remaining structural gap is 1.4, which is nearly 1% of 144. Hence, 15% + 1% baseline maps close to 16% (Precisely 15.97%).

Final Answer: 15.97%

03. Successive Percentage Compounding Shifts

SSC CGL, CPO, TNPSC Group 1, 2 Frequency: Master Class Level

How to Identify This Pattern

Questions tracking values modified continuously across multiple time steps or multi-layered compound paths. Standard phrases include: "increased by A%, then decreased by B%," or consecutive transaction changes.

Three Solving Approaches

a) Traditional Method

Track the variable algebraic value sequentially: Final Value = Initial Value x [1 +/- (a / 100)] x [1 +/- (b / 100)].

b) Traditional Shortcut Method

Use the standard dual-change successive shortcut formula: Net Effective Shift = a + b + (ab / 100). Treat fractional values as negative integers if they represent explicit percentage decreases.

c) Direct Shortcut/Trick

Fractional Ratio Product Method: Convert all percentage modifications directly into scaling structural vectors. Multiply the denominators to find the effective base, and cross-multiply numerators for tracking final system states.

Step-by-Step Approach Summary

  1. Convert all structural percentage values into fractional notation.
  2. Write initial and final states as stacked horizontal columns.
  3. Multiply column values down to arrive at the aggregate system transformation ratio.

Solved Examples

Q1 [Easy]: A salary climbs 20%, then shifts downwards by 10%. Find net change.

Shortcut Solution: Using formula: 20 - 10 + [20 x (-10) / 100] = 10 - 2 = 8%.

Final Answer: 8% Increase

Q2 [Easy]: A retail object value faces two sequential price growth spurts of 10% and 30%. Find net shift.

Shortcut Solution: Net Shift = 10 + 30 + (10 x 30 / 100) = 40 + 3 = 43%.

Final Answer: 43% Increase

Q3 [Moderate]: Car costs rise by 16.67%, then slide down due to flash sales by 14.28%. Calculate net percentage adjustment.

Traditional Solution: Extremely tedious using standard formulas like 16.67 - 14.28 - (16.67 x 14.28 / 100).

Shortcut Solution: Ratio mapping: +16.67% = +1/6 leads to (6 : 7). -14.28% = -1/7 leads to (7 : 6). Multiply compound scales: Initial = 6 x 7 = 42; Final = 7 x 6 = 42. Net transformation scale is exactly zero.

[TIP] The Fractional Ratio Product method converts complex repeating decimals into simple, self-canceling integers.

Final Answer: 0% (No Change)

Q4 [Moderate]: The length of a square expands by 20% while its structural width drops by 20%. Find net structural area delta.

Shortcut Solution: For equal value increases and decreases, apply the rapid structural template - (x^2 / 100). Thus: - (20^2 / 100) = -4%.

Final Answer: 4% Decrease

Q5 [Hard/Twisted]: A factory asset value scales down three times successively by 10%, 20%, and 25%. Calculate absolute net contraction value over the entire operational span.
[TRAP] Do not apply the linear formula pairs sequentially unless you enjoy complex decimal arithmetic.

Traditional Solution: Final = Initial x 0.90 x 0.80 x 0.75 = Initial x 0.54. Contraction scale = 1 - 0.54 = 46%.

Shortcut Solution: Map fractions: 10% = 1/10 leads to (10:9), 20% = 1/5 leads to (5:4), 25% = 1/4 leads to (4:3). Write the operational array:
Initial base total lines: 10 x 5 x 4 = 200
Final output total lines: 9 x 4 x 3 = 108
Net absolute reduction value = 200 - 108 = 92. Percentage contraction ratio = (92 / 200) x 100 = 46%.

Final Answer: 46% Decrease

04. Structural Value Recovery Base Shifts

SSC CGL, CHSL, TNPSC Group 2 Frequency: High

How to Identify This Pattern

A number is modified by a given percentage, and the question asks what percentage the new number must be altered by to recover the exact baseline value.

Three Solving Approaches

a) Traditional Method

Set base variable to 100. Modified state = 100 +/- x. To pull back to original target, solve for required variable shift: Adjustment = [x / (100 +/- x)] x 100.

b) Traditional Shortcut Method

Use inverse functional variation patterns (Product Matrix = Constant Value). If one factor scales up by fraction 1/x, the accompanying balancer structural value must drop by 1 / (x + 1).

c) Direct Shortcut/Trick

Base-Change Rule: If increase is a/b, the required reduction is a / (b + a). If decrease is a/b, the required recovery increase is a / (b - a).

Step-by-Step Approach Summary

  1. Convert the initial percentage change into a simplified fraction a/b.
  2. For an increase, add the numerator to the denominator: a / (b + a).
  3. For a decrease, subtract the numerator from the denominator: a / (b - a).
  4. Convert the resulting fraction back to a percentage.

Solved Examples

Q1 [Easy]: An asset drops by 20%. By what percentage must it rise to hit baseline?

Shortcut Solution: Drop = 20% = 1/5. Recovery fraction layout = 1 / (5 - 1) = 1/4 = 25%.

Final Answer: 25% Increase

Q2 [Easy]: A value scales up by 25%. What percentage reduction recovers original parameters?

Shortcut Solution: Rise = 25% = 1/4. Reduction fraction layout = 1 / (4 + 1) = 1/5 = 20%.

Final Answer: 20% Decrease

Q3 [Moderate]: A worker's hours were cut by 16.67%. By what percentage must his hourly capacity increase to maintain identical output metrics?

Traditional Solution: New baseline value = 100 - 16.67 = 83.33. Required growth calculation = (16.67 / 83.33) x 100 = 20%.

Shortcut Solution: Cut = 16.67% = 1/6. Balance recovery scalar = 1 / (6 - 1) = 1/5 = 20%.

Final Answer: 20%

Q4 [Moderate]: Oil prices soar by 33.33%. Find the percentage consumption reduction required to maintain a constant fuel budget.

Shortcut Solution: Price rise = 33.33% = 1/3. Consumption cut structural factor = 1 / (3 + 1) = 1/4 = 25%.

Final Answer: 25%

Q5 [Hard/Twisted]: Raw input valuation spikes by 37.5%. By what exact fractional percentage value must the processing volume decline so the net operational capital outflow does not scale upwards?
[TRAP] Do not attempt base equations with 137.5 in the denominator.

Traditional Solution: Change calculation = (37.5 / 137.5) x 100 = (3 / 11) x 100 = 27.27%.

Shortcut Solution: Convert to fraction: 37.5% = 3 x 12.5% = 3/8. Since it's an increase, the recovery denominator expands: 3 / (8 + 3) = 3/11. We know 1/11 = 9.09%, so 3/11 = 3 x 9.09% = 27.27%.

💎 Live Gem: Converting non-unit fractions like 3/8 into base-change metrics helps avoid complex long division under pressure.

Final Answer: 27.27% (or 27 and 3/11 %)

05. Asymmetric Comparison Scales

SSC CGL, CHSL, CPO, TNPSC Group 4 Frequency: Constant Appearance

How to Identify This Pattern

Questions comparing two distinct entities, framed as: "If A is X% more than B, then B is what percentage less than A?"

Three Solving Approaches

a) Traditional Method

Fix B = 100. Calculate A = 100 + X. Compute comparative delta difference relative to new base A: Percentage Less = [(A - B) / A] x 100.

b) Traditional Shortcut Method

Construct clear internal dynamic ratio matrices. If A is 20% more than B, then A : B = 120 : 100 which reduces to 6 : 5. Difference is 1 unit, evaluated against A (6), yielding a 1/6 value.

c) Direct Shortcut/Trick

Apply the base-change structure directly: Fraction = Delta / New Denominator Base Line. Scale up uses 1/x to 1/(x+1). Scale down uses 1/x to 1/(x-1).

Step-by-Step Approach Summary

  1. Identify the given percentage comparison and convert it to its fractional form a/b.
  2. Determine the target base (the entity following "than" or "compared to").
  3. Adjust the denominator to form the new fractional relationship, then convert to a percentage.

Solved Examples

Q1 [Easy]: If A's salary is 50% more than B's, how much % is B's salary less than A's?

Shortcut Solution: 50% = 1/2 increase. Reverse fraction balance = 1 / (2 + 1) = 1/3 = 33.33%.

Final Answer: 33.33% less

Q2 [Easy]: If X is 10% less than Y, then Y is what % more than X?

Shortcut Solution: 10% = 1/10 decrease. Reverse fraction balance = 1 / (10 - 1) = 1/9 = 11.11%.

Final Answer: 11.11% more

Q3 [Moderate]: Income of Radha is 14.28% more than Prime. By what percentage is Prime's earnings matrix below Radha's?

Traditional Solution: Let Prime = 100, Radha = 114.28. Delta conversion = (14.28 / 114.28) x 100 = 12.5%.

Shortcut Solution: 14.28% = 1/7. Since it is more, the inverse adjustment fraction is 1 / (7 + 1) = 1/8 = 12.5%.

Final Answer: 12.5%

Q4 [Moderate]: If height of tower P is 37.5% less than tower Q, find how much percent tower Q is taller than P.

Shortcut Solution: 37.5% = 3/8 decrease. Target inverse multiplier layout = 3 / (8 - 3) = 3/5. We know 3/5 = 60%.

Final Answer: 60%

Q5 [Hard/Twisted]: Weight of M is 57.14% more than N. Weight of O is 25% less than the aggregate summation weights of M and N unified. By what percentage value is M's weight less than O's weight?
[TRAP] Do not assign 100 to N. Use the denominator structural value of 7 instead to keep values clean.

Traditional Solution: Complex algebraic scaling using decimal variables.

Shortcut Solution: Recognize 57.14% = 4 x 14.28% = 4/7. Let N = 7 units. Then M = 7 + 4 = 11 units. Total (M + N) = 18 units. Since O is 25% (1/4) less than 18, O = (3/4) x 18 = 13.5 units. The question asks how much less M (11) is than O (13.5): Delta = 13.5 - 11 = 2.5 units. Percentage calculation = (2.5 / 13.5) x 100 = (5 / 27) x 100 = 18.51%.

💎 Live Gem: Defining values based on fractional denominators prevents awkward decimal fractions midway through multi-stage word problems.

Final Answer: 18.51%

06. Compound Growth & Depreciation Architecture

SSC CGL, CPO, TNPSC Group 1, 2, 4 Frequency: High Priority

How to Identify This Pattern

Questions framing population expansion or machine depreciation tracking across multiple years. Look for key indicator terms like "compounded annually" or "depreciates at R% per annum."

Three Solving Approaches

a) Traditional Method

Apply the geometric growth formula: V_final = V_initial x [1 +/- (R / 100)]^n, where n represents the time step horizon.

b) Traditional Shortcut Method

Apply the successive percentage compounding formula step-by-step [a + b + (ab / 100)] to determine the net percentage multiplier across the target timeline.

c) Direct Shortcut/Trick

Ratio Squaring/Cubing Strategy: Convert the rate to a simplified fraction. For a 3-year timeline, cube the ratio limits directly: (Base Unit)^3 to (Modified Unit)^3.

Step-by-Step Approach Summary

  1. Convert the growth or depreciation rate into its fractional equivalent.
  2. Write the single-year transition ratio (Original : New).
  3. Raise the ratio terms to the power of n (where n is the number of years).
  4. Map the scaled units directly to the given numerical values to solve.

Solved Examples

Q1 [Easy]: A town population grows at 10% annually. Current population is 10000. Find population after 2 years.

Shortcut Solution: Rate = 10% = 1/10 which implies 10 : 11. Square for 2 years: 10^2 : 11^2 = 100 : 121. If 100 units = 10000, then 1 unit = 100. Output population = 121 x 100 = 12100.

Final Answer: 12,100

Q2 [Easy]: A machine costs ₹8000. It depreciates by 10% every year. Find its value after 2 years.

Shortcut Solution: Depreciation = 1/10 which implies 10 : 9. For 2 years: 100 : 81. Given 100 units = 8000 implying 1 unit = 80. Residual machine value = 81 x 80 = 6480.

Final Answer: ₹6,480

Q3 [Moderate]: A structural component value drops 20% annually. If current valuation sits at ₹10,240, what was its initial value 3 years ago?

Traditional Solution: 10240 = V x (0.80)^3 implying 10240 = V x 0.512, which resolves to V = 10240 / 0.512 = 20000.

Shortcut Solution: Drop = 20% = 1/5 which implies 5 : 4. Cube for 3 years: 5^3 : 4^3 = 125 : 64. Given the final state 64 units = 10240 implying 1 unit = 10240 / 64 = 160. Initial value = 125 x 160 = 20000.

Final Answer: ₹20,000

Q4 [Moderate]: The population of a village increases by 5% in the first year and drops by 5% in the second year. If the current population is 39,900, find the population two years ago.

Shortcut Solution: Year 1: +5% = +1/20 which maps to 20 : 21. Year 2: -5% = -1/20 which maps to 20 : 19. Combined compound ratio: 20 x 20 : 21 x 19 = 400 : 399. Given 399 units = 39900 implying 1 unit = 100. Initial population value = 400 x 100 = 40000.

Final Answer: 40,000

Q5 [Hard/Twisted]: A smart system value degrades by 12.5% in year 1, grows by 22.22% in year 2, and drops by 14.28% in year 3. If the valuation asset scale after 3 years is ₹6,600, calculate the original valuation baseline.
[TRAP] Trying to solve this with standard decimal formulas will waste valuable time. Convert to fractions immediately.

Traditional Solution: Extremely complex chain equations using decimals like V x 0.875 x 1.2222 x 0.8571 = 6600.

Shortcut Solution: Convert all changes to fractions:
Year 1: -12.5% = -1/8 leads to (8 : 7)
Year 2: +22.22% = +2/9 leads to (9 : 11)
Year 3: -14.28% = -1/7 leads to (7 : 6)
Multiply the ratios: (8 x 9 x 7) / (7 x 11 x 6). Cancel common terms across the ratio columns: the 7s cancel out, and 9 and 6 simplify by 3. This gives an aggregate scale ratio of 16 : 11.
Given final state 11 units = 6600 implying 1 unit = 600. Initial state baseline = 16 x 600 = 9600.

💎 Live Gem: Cross-canceling terms across initial and final ratio columns simplifies complex multi-year calculations.

Final Answer: ₹9,600

07. Budget Allocation Matrices (Income/Expenditure)

SSC CGL, CHSL, CPO, TNPSC Group 1, 2 Frequency: Very High

How to Identify This Pattern

Questions that track a person's income distribution, framed using either raw percentage allocations (e.g., "spends 20% on rent, 30% on food") or sequential remainder allocations (e.g., "spends 20% of the *remaining* balance").

Three Solving Approaches

a) Traditional Method

Set absolute Income to variable I. Construct a linear algebraic balance sheet equation: Income - (Expense_1 + Expense_2) = Savings. For remainder problems, use cascading parenthetical terms: I x (1 - a) x (1 - b) = Savings.

b) Traditional Shortcut Method

Assume Income base = 100. Subtract expenditures sequentially using a step-by-step tree diagram to preserve the distinct structural boundaries of each spending category.

c) Direct Shortcut/Trick

For sequential remainder problems, apply the Fractional Compounding method. Multiply the reciprocal fractions of the remaining balance to calculate total income in a single step.

Step-by-Step Approach Summary

  1. Check if expenditures apply to total income (flat addition) or the remaining balance (successive multiplication).
  2. For remainder problems, convert the spent percentages into net remaining fractional units.
  3. Set up the product sequence: Total Income = Given Savings x Product of (Denominator / Remaining Numerator) terms.

Solved Examples

Q1 [Easy]: A man earns money, spends 75% of it, and saves ₹3000. Calculate his total income.

Shortcut Solution: Spending = 75% implying Savings = 25% = 1/4. If (1/4) x Income = 3000, then Income = 12000.

Final Answer: ₹12,000

Q2 [Easy]: An employee spends 20% on fuel and 40% on food out of total income. If he saves ₹4000, find his income.

Shortcut Solution: Total expenses are flat additions: 20% + 40% = 60%. Savings score = 100% - 60% = 40%. If 40% = 4000, then 100% = 10000.

Final Answer: ₹10,000

Q3 [Moderate]: A manager spends 20% of his income on house rent, and 70% of the remainder on children's education. If he saves ₹3,600, find his total income.

Traditional Solution: Let Income = X. Rent expense = 0.20X, leaving a remainder of 0.80X. Education expense = 0.70 x 0.80X = 0.56X. Final savings balance = 0.80X - 0.56X = 0.24X. Set 0.24X = 3600 which updates to X = 3600 / 0.24 = 15000.

Shortcut Solution: Convert to remaining fractions: Rent spent = 20% = 1/5 implies Left = 4/5. Education spent = 70% = 7/10 implies Left = 3/10. Total Income layout = 3600 x (5 / 4) x (10 / 3) = 300 x 50 = 15000.

Final Answer: ₹15,000

Q4 [Moderate]: A person's income increases by 20% and expenditure increases by 10%. If his original income was ₹20,000 and savings were ₹8,000, find the percentage increase in his savings.

Shortcut Solution: Set up a linear scale components grid:
Original: Income (200) = Expense (120) + Savings (80).
New values: Income increases by 20% implying 200 x 1.2 = 240. Expense increases by 10% implying 120 x 1.1 = 132.
New Savings balance = 240 - 132 = 108. Increase in savings value = 108 - 80 = 28 units. Percentage change = (28 / 80) x 100 = 35%.

Final Answer: 35%

Q5 [Hard/Twisted]: A man spends 14.28% of his monthly income on medical costs, 25% of the remainder on house rent, and 33.33% of the next remainder on food. The final remaining balance is split between savings and entertainment in a 5 : 3 ratio. If his entertainment spend is ₹1,500, calculate his total monthly income.
[TRAP] Do not attempt to use 100 as a base value here. The fractions will quickly become messy.

Traditional Solution: Multi-layered algebraic variable strings that require significant computation time.

Shortcut Solution: First, find the total balance before the final split. The entertainment allocation is 3 parts out of an 8-part total (5 + 3). If 3 units = 1500, then 1 unit = 500, making the total balance before the split 8 x 500 = 4000. Now, work backward using the remaining fractions:
Medical cost = 14.28% = 1/7 implies Left fraction = 6/7
Rent cost = 25% = 1/4 implies Left fraction = 3/4
Food cost = 33.33% = 1/3 implies Left fraction = 2/3
Calculate Total Income by multiplying by the reciprocal fractions: Income = 4000 x (7 / 6) x (4 / 3) x (3 / 2). The 3s cancel out, and the product grid simplifies neatly to: Income = 4000 x (7 / 4) = 7000.

💎 Live Gem: Converting final ratio balances into working integers allows you to trace fractions backward cleanly without decimal clutter.

Final Answer: ₹7,000

08. Bivariate Voting Thresholds (Elections)

SSC CGL, CPO, TNPSC Group 1, 2 Frequency: High Priority

How to Identify This Pattern

Questions tracking election votes between candidates. Look for complicating constraints like "10% of voters did not cast votes," "10% of votes were declared invalid," and "the winner won by a majority of valid votes."

Three Solving Approaches

a) Traditional Method

Let total registered voters = X. Deduct non-voting lines: Cast = 0.90X. Deduct invalid votes: Valid = 0.90 x 0.90X. Set up the candidate vote differential equation: (Winner % - Loser %) x Valid Votes = Winning Margin.

b) Traditional Shortcut Method

Assume the total electorate baseline = 100 units. Work down the voting tree step-by-step to compute the net valid vote pool before allocating candidate shares.

c) Direct Shortcut/Trick

Chain Fraction Equation: Total Electorate = Margin x [100 / (Winner % - Loser %)] x Product of (Total Ratio / Valid Ratio) remaining vectors.

Step-by-Step Approach Summary

  1. Find the net percentage gap between the winning and losing candidates based on valid votes.
  2. Convert all reductions (non-voters, invalid votes) into remaining fractional multipliers.
  3. Set up a single chain equation starting from the winning margin to find the total voter base.

Solved Examples

Q1 [Easy]: In a straight two-candidate election, the winner gets 60% of the votes and wins by a margin of 2000 votes. Find total votes.

Shortcut Solution: Winner = 60% implies Loser = 40%. The percentage gap (Delta) = 20%. If 20% = 2000, then 100% = 10000.

Final Answer: 10,000

Q2 [Easy]: 10% of voters did not cast their votes in an election. The winner secured 60% of the cast votes and won by 1800 votes. Find total list voters.

Shortcut Solution: Gap within cast votes pool = 60% - 40% = 20%. Cast votes amount = 1800 x (100 / 20) = 9000. Since 10% did not vote, the cast pool is 9/10 of the total. Total voters scale = 9000 x (10 / 9) = 10000.

Final Answer: 10,000

Q3 [Moderate]: An election records 10% non-voting lists. Out of the cast votes, 600 votes are declared invalid. The winning candidate secures 75% of the valid votes and wins by a margin of 1500 votes. Find the total number of registered voters.

Traditional Solution: Let Total = X. Cast = 0.9X. Valid = 0.9X - 600. Winner gets 75%, Loser gets 25% implying Gap = 50% = 0.5. Equation: 0.5 x (0.9X - 600) = 1500 leading to 0.9X - 600 = 3000, which factors out to 0.9X = 3600, hence X = 4000.

Shortcut Solution: Candidate percentage gap = 75% - 25% = 50% = 1/2. Total valid votes pool = 1500 x (2 / 1) = 3000. Add the invalid votes back to find cast votes: 3000 + 600 = 3600. Since 10% did not vote, 3600 represents 9/10 of the total list. Total Electorate base = 3600 x (10 / 9) = 4000.

Final Answer: 4,000

Q4 [Moderate]: In an election, 20% of voters did not vote, and 10% of the cast votes were invalid. The winning candidate received 60% of the valid votes and won by a majority of 2880 votes. Find the total number of registered voters.

Shortcut Solution: Candidate percentage gap = 60% - 40% = 20% = 1/5. Remainder multipliers: Non-voters = 20% = 1/5 implies Left fraction = 4/5. Invalid votes = 10% = 1/10 implies Left fraction = 9/10. Set up the calculation chain: Total Electorate = 2880 x (5 / 1) x (10 / 9) x (5 / 4). This simplifies systematically to 80 x 250 = 20000.

Final Answer: 20,000

How to Identify This Pattern

Questions that combine percentage metrics with cost price and selling price updates. For example: "If Cost Price increases by 20% and Selling Price decreases by 10%, the profit shifts from 20% to its new value."

Three Solving Approaches

a) Traditional Method

Assign initial variables: Cost Price = CP, Selling Price = SP = CP x (1 + P_1). Calculate the updated states: CP_new = CP x (1 + alpha), SP_new = SP x (1 - beta). Calculate new profit using: [(SP_new - CP_new) / CP_new] x 100.

b) Traditional Shortcut Method

Set the initial Cost Price baseline to 100. Calculate the initial Selling Price based on the original profit margin. Apply the given percentage changes to both values to find the new profit margin.

c) Direct Shortcut/Trick

Ratio Grid Factorisation: Convert the cost and selling price adjustments into multipliers, and calculate the new profit margin directly from the updated ratio matrix.

Step-by-Step Approach Summary

  1. Set the original Cost Price (CP) to 100 and determine the original Selling Price (SP).
  2. Apply the specified percentage updates to calculate the new CP and SP.
  3. Find the new profit margin by calculating the difference between the updated values relative to the new CP.

Solved Examples

Q1 [Easy]: An item yields a 20% profit. If its Cost Price increases by 10%, find the new profit percentage assuming the Selling Price remains unchanged.

Shortcut Solution: Let initial CP = 100 implies SP = 120. Updated CP = 110, while SP remains 120. New profit score = 120 - 110 = 10 units. Percentage calculation = (10 / 110) x 100 = 9.09%.

Final Answer: 9.09% Profit

Q2 [Easy]: A trader makes a 10% profit. If he doubles his Selling Price, calculate his new profit percentage.

Shortcut Solution: Let CP = 100 implies SP = 110. Doubling the selling price gives a new SP = 220. New profit margin = 220 - 100 = 120 units, which means a 120% profit margin.

Final Answer: 120% Profit

Q3 [Moderate]: An article is sold at a profit of 25%. If both its cost price and selling price decrease by ₹20, the profit margin increases by 5%. Find the original cost price.

Traditional Solution: Let CP = 4x implies SP = 5x. New system equation: (5x - 20 - (4x - 20)) / (4x - 20) = 30% which shifts to x / (4x - 20) = 3/10, hence 10x = 12x - 60, resolving to 2x = 60, so 4x = 120.

Shortcut Solution: Initial profit = 25% = 1/4 implies ratio is 4 : 5. New profit = 25% + 5% = 30% = 3/10 implies ratio is 10 : 13. To balance the identical change (₹20 reduction on both sides), ensure the difference between the ratio terms is equal. Multiply the first ratio row by 3: 12 : 15. The second ratio remains 10 : 13. The structural difference on both sides is exactly 2 units (12 - 10 and 15 - 13). If 2 units = 20 rupees, then 1 unit = 10 rupees. Original CP base = 12 units = 120 rupees.

[TIP] Balancing ratio differences avoids setting up complex algebraic equations for identical value changes.

Final Answer: ₹120

Q4 [Moderate]: A merchant sells goods at a 20% profit margin. If the cost price rises by 15% and he scales up the selling price by 10%, find his new profit percentage.

Shortcut Solution: Let initial CP = 100 implies SP = 120. Updated CP = 115. Updated SP = 120 x 1.10 = 132. New profit amount = 132 - 115 = 17 units. Profit percentage layout = (17 / 115) x 100 = 14.78%.

Step-by-Step Approach Summary

How to Identify This Pattern

Questions where the price of a commodity changes, and the user must find the required change in consumption to keep the total expenditure constant.

Three Solving Approaches

a) Traditional Method

Use the base product formula: Price x Consumption = Expenditure. If price changes by P%, set up the balanced equation: 1.00 x 1.00 = [1 + (P / 100)] x [1 - (C / 100)]. Solve for C.

b) Traditional Shortcut Method

Set initial price and consumption values to 10 each, making the original expenditure 100. Apply the price change, then adjust consumption to return the product to 100.

c) Direct Shortcut/Trick

Base-Change Reciprocal Rule: An increase of 1/x in price requires a 1/(x+1) reduction in consumption. A decrease of 1/x in price requires a 1/(x-1) increase in consumption.

Step-by-Step Approach Summary

  1. Convert the percentage change in price into a simplified fraction a/b.
  2. For a price increase, the consumption reduction fraction is a / (b + a).
  3. For a price decrease, the consumption increase fraction is a / (b - a).
  4. Convert the resulting fraction back into a percentage.

Solved Examples

Q1 [Easy]: The price of sugar rises by 25%. Find the percentage reduction in consumption required to keep expenditure the same.

Shortcut Solution: Price increase = 25% = 1/4. Required consumption reduction vector = 1 / (4 + 1) = 1/5 = 20%.

Q2 [Easy]: The price of petrol drops by 20%. By how much percent can a driver increase consumption without changing their fuel budget?

Shortcut Solution: Use ratios: Price increases by 33.33% = 1/3 implies price ratio is 3 : 4. Expenditure increases by 10% implies expenditure ratio is 10 : 11. Since Consumption = Expenditure / Price, the consumption ratio layout scales out to (10 / 3) : (11 / 4) = 40 : 33. The reduction in consumption is 7 units on a base of 40: (7 / 40) x 100 = 17.5%.

[TIP] When expenditure is not constant, dividing the expenditure ratio by the price ratio gives the consumption ratio directly.

Final Answer: 17.5%

Q4 [Moderate]: A price reduction of 20% enables a buyer to purchase 5 kg more of an item for ₹100. Find the original price per kg.

Shortcut Solution: Price reduction = 20% = 1/5 implies consumption increases by 1 / (5 - 1) = 1/4. This 1/4 increase in consumption equals 5 kg, so the original consumption was 4 x 5 = 20 kg. Original price rate = ₹100 / 20 kg = ₹5 per kg.

Final Answer: ₹5 per kg

Q5 [Hard/Twisted]: The price of an item drops by 14.28%. This change allows a consumer to buy 4 more items for ₹560, while also saving 10% of their original budget for other expenses. Calculate the updated price per item.
[TRAP] Keep track of the two changes: the price drop increases purchasing power, but the reduced budget reduces it.

Traditional Solution: Setting up equations with multiple variables that require lengthy calculation to solve.

Shortcut Solution: Price drop = 14.28% = 1/7 implies price ratio is 7 : 6. The budget is reduced by 10%, so the expenditure ratio is 10 : 9. Find the consumption ratio by dividing expenditure by price: (10 / 7) : (9 / 6) = (10 / 7) : (3 / 2) = 20 : 21. The increase in consumption is 1 unit. We are given that this 1 unit equals 4 items, so the new consumption quantity is 21 x 4 = 84 items. The new expenditure amount is 90% of ₹560 = ₹504. Updated price per item = 504 / 84 = ₹6.

💎 Live Gem: Dividing the expenditure ratio by the price ratio works reliably even when both budget constraints and prices change at the same time.

Shortcut Solution: Calculate the total percentage of students who passed at least one subject: Total Pass Union = 50% + 60% - 20% = 90%. The percentage of students who failed both subjects is the remainder: 100% - 90% = 10%.

Final Answer: 10% Failed Both

Q5 [Hard/Twisted]: In a survey, 70% of respondents own a car, and 75% own a house. If 15% own neither a car nor a house, find the exact percentage of respondents who own both assets.

Shortcut Solution: Since 15% own neither, the total percentage of respondents who own at least one asset is 100% - 15% = 85%. Now use the set union formula to find the shared intersection (those who own both): Car + House - Both = Total Union implying 70% + 75% - Both = 85%, which moves to 145% - Both = 85%. Solving for both gives: Both = 145% - 85% = 60%.

💎 Live Gem: Ensure all data points are converted to the same metric boundary (either all participating/passing or all non-participating/failing) before applying the union formula.

Final Answer: 60% Own Both

20. Product-Stability Structural Shift

SSC CGL Master Framework Frequency: High Difficulty Level

How to Identify This Pattern

Advanced questions where two related factors change at the same time, shifting the total output product. For example, tracking changes in total revenue when both admission ticket prices and visitor attendance numbers change simultaneously.

Three Solving Approaches

a) Traditional Method

Set up the full product equation: Product_new = [Factor_1 x (1 + alpha)] x [Factor_2 x (1 - beta)]. Expand the expression algebraically to find the net change.

b) Traditional Shortcut Method

Apply the successive percentage formula: Net Shift = a + b + (ab / 100), treating reductions as negative numbers.

c) Direct Shortcut/Trick

Fractional Ratio Matrix: Convert both percentage changes into fractional ratios, then multiply the ratios vertically to find the net change in the total product directly.

Solved Examples (Shortcut Driven)

Q1 [Easy]: Cinema ticket prices rise by 20%, causing attendance to drop by 10%. Find the net percentage change in total revenue.

Shortcut Solution: Apply the successive change formula: 20 - 10 + (20 x (-10) / 100) = 10 - 2 = 8%.

Final Answer: 8% Increase in Revenue

Q5 [Hard/Twisted]: A factory increases its working hours per day by 25%, while upgrading its machinery to improve hourly production efficiency by 16.67%. At the same time, raw material defects cause a 10% loss in total usable output. Calculate the net percentage change in the factory's final usable production volume.

Shortcut Solution: Convert all three changes into fractional ratios:
Working hours: +25% = +1/4 implies 4 : 5 ratio row
Efficiency: +16.67% = +1/6 implies 6 : 7 ratio row
Material defects: -10% = -1/10 implies 10 : 9 ratio row
Multiply the ratios together: Initial Base = 4 x 6 x 10 = 240; Final Output = 5 x 7 x 9 = 315.
Simplify the structural scale fraction: 315 / 240 = 21 / 16. The net increase is 5 parts on a base of 16: (5 / 16) x 100 = 31.25%.

💎 Live Gem: The Fractional Ratio Matrix tracks multiple simultaneous changes smoothly, avoiding the errors that can occur when applying the linear formula repeatedly.

Final Answer: 31.25% Increase

Common Pitfalls & Traps Used by Examiners

The Base Shift Trap: Examiners often frame candidate vote shares around the total registered electorate instead of the valid vote pool. Always read the wording carefully to identify the correct base.
Prepositional Traps: Watch for the difference between "increased by 20%" (adding 20% to the base) and "increased to 120%" (the final value becomes 120%). Misinterpreting these words changes the equation entirely.
Percentage Points vs. Percentages: A change from 10% to 15% is a 5 percentage point increase, but a 50% percentage increase in the rate itself. Double-check the required units in the question.
